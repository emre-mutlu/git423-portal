# Neden Kod Bilmeliyiz?

Bir görsel iletişim tasarımcısı olarak sadece arayüzün "nasıl göründüğünü" değil, "nasıl çalıştığını" ve "tarayıcının onu nasıl okuduğunu" bilmek sizi sektörde rakiplerinizden ayırır. Kod bilmek, tasarımlarınızın sınırlarını genişletir.

## Çalışma Ekosistemi

Bu derste kullanacağımız temel araçlar:
1. **Visual Studio Code (VS Code):** Dünyanın en popüler ve genişletilebilir kod editörü.
2. **GitHub & GitHub Classroom:** Ödev teslimlerimizi ve kod versiyonlamamızı sağlayacak endüstri standardı platform.
3. **AI Asistanları:** Kod yazarken karşılaştığınız hataları (debugging) anlamak ve öğrenme hızınızı artırmak için yapay zekayı bir "mentor" gibi kullanmayı öğreneceğiz.

## Bu Haftanın Görevi
VS Code kurulumunuzu tamamlayın, GitHub hesabı oluşturun ve ilgili Classroom linkine tıklayarak ilk deponuzu (repository) onaylayın.
